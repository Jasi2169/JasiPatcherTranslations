!!..Troubleshooting..!!
 -> Update SuperSU SUPERSU

 -> Install Busybox BUSYBOX

 -> Apply Patch On Untouched Or Unmodified APKs Only,Patching Already Modified APKs Will Result In Not Working,If You Need PAID APKs,Search 4PDA Forums !!

  -> Make Sure Mount Namespace Separation Option Is Unchecked In SuperSU Settings !!

 -> Make Sure You Do Not Install App In System OR Move Your App To External SDCard Otherwise Patch Will Fail !!

 -> If Result Shows Patched But App Is Not Patched,Make Sure Play Store Account Is Login In Device !!

 -> If Result Shows Patched But App Is Not Patched,Try Patching After Reboot (Only Happen In Rare Cases When Resources Are Busy) !!
